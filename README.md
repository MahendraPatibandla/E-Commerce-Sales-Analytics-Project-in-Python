## 📑 Overview

A comprehensive data analysis project focused on uncovering actionable insights from an e-commerce superstore dataset. This project demonstrates end-to-end data analysis skills including data cleaning, exploratory data analysis (EDA), and interactive visualization to drive business decisions.

## 🎯 Project Objectives

- **Analyze Sales Performance** across different customer segments, regions, and product categories
- **Identify Profitability Trends** to optimize business strategy
- **Discover Regional Patterns** to understand geographic performance variations
- **Evaluate Product Performance** to determine best and worst performing categories
- **Calculate Key Metrics** including Sales-to-Profit ratios for strategic planning

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **Python 3.13.5** | Core programming language |
| **Pandas** | Data manipulation and analysis |
| **Plotly Express** | Interactive data visualizations |
| **Plotly Graph Objects** | Advanced chart customization |
| **Jupyter Notebook** | Development environment |

## 📂 Dataset Information

- **Source**: Superstore Sales Dataset
- **Records**: 9,994 transactions
- **Features**: 21 columns
- **Key Variables**:
  - 📦 Order Details (Order ID, Date, Ship Mode)
  - 👤 Customer Information (ID, Name, Segment)
  - 📍 Geographic Data (Country, State, City, Region, Postal Code)
  - 🛍️ Product Details (Category, Sub-Category, Product Name)
  - 💰 Financial Metrics (Sales, Profit, Discount, Quantity)

## 🔍 Key Analyses Performed

### 1️⃣ **Sales and Profit by Category**
- Analyzed performance across Furniture, Office Supplies, and Technology
- Visualized revenue vs profitability patterns
- Identified high-revenue but low-margin categories

### 2️⃣ **Regional Performance Analysis**
- Compared sales and profit metrics across Central, East, South, and West regions
- Created interactive bar charts for regional comparison
- Discovered regional strengths and opportunities

### 3️⃣ **Customer Segment Analysis**
- Evaluated Consumer, Corporate, and Home Office segments
- Calculated Sales-to-Profit ratios for each segment
- **Key Finding**: 
  - Consumer: 8.66 ratio
  - Corporate: 7.68 ratio
  - Home Office: 7.13 ratio (most efficient)

### 4️⃣ **Product Category Deep Dive**
- Examined sub-category performance
- Identified top-performing and loss-making products
- Provided recommendations for inventory optimization

### 5️⃣ **Shipping Mode Analysis**
- Evaluated different shipping methods
- Analyzed relationship between shipping mode and profitability

## 💡 Key Insights

✅ **Home Office segment** shows the best Sales-to-Profit efficiency (7.13 ratio)  
✅ **Consumer segment** generates highest revenue but with lower efficiency (8.66 ratio)  
✅ **Regional variations** suggest opportunities for targeted marketing  
✅ **Category-level analysis** reveals strategic focus areas for profitability
